## 1. Introduccion

El modulo de **Aseo Contratado** gestiona los contratos de servicio de recoleccion de basura para comercios y empresas del Municipio de Guadalajara. Este modulo permite:

- Administracion de catalogos (empresas, zonas, tipos de aseo)
- Alta, modificacion y baja de contratos
- Control de adeudos y pagos
- Aplicacion de multas y descuentos
- Generacion de reportes y estados de cuenta

## 8. Glosario

| Termino | Definicion |
|---------|------------|
| **Contrato** | Acuerdo de servicio de recoleccion |
| **Zona** | Area geografica de cobertura |
| **Unidad** | Vehiculo de recoleccion |
| **Adeudo** | Monto pendiente de pago |
| **Recargo** | Cargo adicional por mora |
| **Condonacion** | Perdon de deuda autorizado |
| **Ejercicio** | Ano fiscal de referencia |
| **Recaudadora** | Punto de cobro autorizado |
| **Tarifa** | Costo del servicio |

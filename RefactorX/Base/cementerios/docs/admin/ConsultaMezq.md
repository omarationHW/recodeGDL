# ConsultaMezq - Consulta de Panteón Mezquitán

## Propósito Administrativo
Módulo de consulta especializado para el Panteón Municipal de Mezquitán, uno de los cementerios históricos más importantes de Guadalajara.

## Funcionalidad Principal
Consulta filtrada de espacios funerarios del Panteón Mezquitán con todas las funcionalidades de búsqueda, reportes y estadísticas específicas de este cementerio.

## Proceso de Negocio

### ¿Qué hace?
- Consultas exclusivas del Panteón Mezquitán
- Búsqueda por ubicación física dentro del panteón
- Reportes históricos del cementerio
- Control de espacios disponibles
- Estadísticas de ocupación

### ¿Para qué sirve?
- Administración del Panteón Mezquitán
- Atención especializada a usuarios de este panteón
- Control de registros históricos
- Generación de reportes administrativos

## Datos y Tablas
**ta_13_datosrcm** - Filtrada por cementerio="MEZQ"

## Usuarios del Sistema
- Administradores del Panteón Mezquitán
- Personal administrativo del cementerio
- Directivos para consultas específicas

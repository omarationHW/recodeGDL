# ConsultaRCM - Consulta General por Criterios RCM

## Propósito Administrativo
Módulo de consulta general que permite búsquedas combinando múltiples criterios: cementerio, ubicación física y características del espacio.

## Funcionalidad Principal
Búsqueda flexible con múltiples criterios combinados, útil para análisis y reportes con filtros complejos.

## Datos y Tablas
**ta_13_datosrcm** - Consulta con múltiples criterios AND/OR

## Usuarios del Sistema
Analistas, personal de estadística, directivos

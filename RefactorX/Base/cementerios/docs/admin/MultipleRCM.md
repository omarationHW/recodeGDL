# MultipleRCM - Consulta Múltiple por Ubicación

## Propósito Administrativo
Módulo de consulta que permite buscar múltiples registros de espacios funerarios utilizando criterios de ubicación física dentro de los panteones.

## Funcionalidad Principal
Realiza búsquedas masivas de espacios por cementerio, clase, sección, línea y fosa, mostrando listados de registros que cumplan los criterios especificados.

## Datos y Tablas
**ta_13_datosrcm** - Búsqueda con múltiples criterios de ubicación

## Usuarios del Sistema
Personal de ventanilla, supervisores, administradores de panteones

# TrasladoFol - Traslados por Folio

## Propósito Administrativo
Módulo para registrar traslados de restos utilizando directamente los folios RCM de origen y destino, agilizando el proceso cuando se conocen los números de folio.

## Funcionalidad Principal
Registra traslados de restos entre espacios usando folios RCM, validando disponibilidad, registrando pago del servicio y actualizando registros.

## Datos y Tablas
Similar a Traslados pero con búsqueda directa por folio

## Usuarios del Sistema
Personal de panteones, supervisores de traslados

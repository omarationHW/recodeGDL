# Adeudos

## Descripción
Módulo para la gestión y consulta de adeudos del sistema de aseo contratado (recolección de basura/limpieza urbana).

## Funcionalidades
- Consulta de adeudos por contrato
- Visualización de montos pendientes
- Historial de adeudos
- Filtros de búsqueda y consulta
- Exportación de información

## Campos
- **Número de Contrato**: Identificador del contrato
- **Nombre/Razón Social**: Datos del contribuyente
- **Monto Adeudo**: Cantidad pendiente de pago
- **Periodo**: Periodo correspondiente al adeudo
- **Fecha de Vencimiento**: Plazo límite de pago

## Instrucciones de Uso
1. Acceda al módulo desde el menú principal de Aseo Contratado
2. Utilice los filtros para buscar contratos específicos
3. Revise la información de adeudos en la tabla de resultados
4. Puede exportar la información para análisis externo

## Notas Importantes
- Este módulo es de consulta, no permite modificaciones directas
- Para pagos, utilice el módulo correspondiente de aplicación de pagos
- La información se actualiza automáticamente desde la base de datos
- Mantiene histórico completo de adeudos

# AplicaMultas

## Descripción
Módulo para la aplicación de multas en el sistema de aseo contratado.

## Funcionalidades
- Aplicación de multas a contratos
- Registro de infracciones
- Cálculo automático de montos según catálogo
- Consulta de multas aplicadas
- Generación de documentación

## Campos
- **Número de Contrato**: Contrato al que se aplica la multa
- **Tipo de Multa**: Catálogo de infracciones
- **Monto**: Cantidad de la multa
- **Fecha de Aplicación**: Cuando se registró la multa
- **Observaciones**: Detalles de la infracción

## Instrucciones de Uso
1. Seleccione el contrato al que aplicará la multa
2. Elija el tipo de multa del catálogo
3. Verifique el monto calculado automáticamente
4. Capture las observaciones necesarias
5. Confirme la aplicación de la multa

## Notas Importantes
- Las multas afectan directamente los adeudos del contrato
- Requiere permisos especiales para aplicar multas
- Genera registros de auditoría
- Una vez aplicada, requiere autorización para cancelarse

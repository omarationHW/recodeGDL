# Cons_ContAsc

## Descripción
Módulo de consulta de contratos en orden ascendente del sistema de aseo contratado.

## Funcionalidades
- Consulta de contratos ordenados ascendentemente
- Filtros de búsqueda
- Visualización detallada de información
- Exportación de datos
- Navegación por registros

## Campos
- **Número de Contrato**: Identificador
- **Nombre**: Titular del contrato
- **Dirección**: Ubicación del servicio
- **Fecha de Alta**: Cuando se registró
- **Estatus**: Estado actual del contrato

## Instrucciones de Uso
1. Ingrese al módulo de consultas
2. Los contratos se mostrarán en orden ascendente
3. Utilice los filtros para refinar la búsqueda
4. Seleccione un contrato para ver detalles
5. Exporte si necesita análisis externo

## Notas Importantes
- Vista de solo consulta
- Ordenamiento automático ascendente
- Actualización en tiempo real
- Integrado con otros módulos del sistema

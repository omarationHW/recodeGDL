# Contratos_Alta

## Descripción
Módulo para dar de alta nuevos contratos en el sistema de aseo contratado.

## Funcionalidades
- Registro de nuevos contratos
- Captura de datos del contribuyente
- Asignación de tipo de servicio
- Definición de periodicidad de pago
- Cálculo de cuotas
- Generación de número de contrato

## Campos
- **Datos del Contribuyente**: Nombre, RFC, domicilio
- **Tipo de Servicio**: Catálogo de servicios de aseo
- **Periodicidad**: Mensual, bimestral, etc.
- **Unidades de Recolección**: Cantidad de unidades
- **Zona**: Área geográfica del servicio
- **Fecha de Inicio**: Vigencia del contrato

## Instrucciones de Uso
1. Acceda al módulo de Altas
2. Capture los datos del contribuyente
3. Seleccione el tipo de servicio del catálogo
4. Defina la periodicidad y unidades
5. Verifique el cálculo de cuotas
6. Confirme el alta del contrato
7. Imprima el contrato generado

## Notas Importantes
- Genera número de contrato automáticamente
- Valida datos antes de guardar
- Requiere permisos de captura
- Crea registros en múltiples tablas
- Genera adeudos iniciales si aplica

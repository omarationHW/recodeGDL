# Contratos_Baja

## Descripción
Módulo para dar de baja contratos existentes en el sistema de aseo contratado.

## Funcionalidades
- Cancelación de contratos
- Validación de adeudos pendientes
- Registro de motivo de baja
- Generación de finiquito
- Actualización de estatus

## Campos
- **Número de Contrato**: Contrato a dar de baja
- **Motivo de Baja**: Razón de la cancelación
- **Fecha de Baja**: Cuando se efectúa
- **Adeudos Pendientes**: Validación de saldos
- **Observaciones**: Notas adicionales

## Instrucciones de Uso
1. Busque el contrato a dar de baja
2. Verifique que no tenga adeudos pendientes
3. Capture el motivo de baja
4. Confirme la fecha de cancelación
5. Genere el documento de finiquito si aplica
6. Confirme la baja del contrato

## Notas Importantes
- No permite baja si existen adeudos
- Requiere autorización especial
- Genera registro de auditoría
- Puede requerirse finiquito
- La baja no elimina el historial

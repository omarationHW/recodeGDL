# Contratos_Mod

## Descripción
Módulo para modificar datos de contratos existentes en el sistema de aseo contratado.

## Funcionalidades
- Actualización de datos del contribuyente
- Cambio de domicilio de servicio
- Modificación de tipo de servicio
- Ajuste de unidades de recolección
- Actualización de periodicidad
- Registro de cambios

## Campos
- **Número de Contrato**: Identificador del contrato
- **Datos Modificables**: Nombre, domicilio, teléfono
- **Tipo de Servicio**: Puede actualizarse
- **Unidades**: Incremento o reducción
- **Zona**: Cambio de área geográfica
- **Motivo del Cambio**: Justificación

## Instrucciones de Uso
1. Busque el contrato a modificar
2. Visualice los datos actuales
3. Modifique los campos necesarios
4. Capture el motivo del cambio
5. Verifique el impacto en cuotas si aplica
6. Confirme los cambios
7. Imprima constancia de modificación si es necesario

## Notas Importantes
- Algunos cambios afectan el cálculo de cuotas
- Requiere permisos especiales
- Mantiene historial de modificaciones
- Genera registro de auditoría
- Cambios mayores pueden requerir autorización

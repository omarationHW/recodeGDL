# Rpt_Adeudos

## Descripción
Reporte general de adeudos del sistema de aseo contratado.

## Funcionalidades
- Generación de reporte de adeudos
- Filtrado por periodo, zona, empresa
- Agrupación por diferentes criterios
- Totales y subtotales
- Exportación a Excel/PDF
- Impresión de reportes

## Campos
- **Número de Contrato**: Identificador
- **Nombre/Razón Social**: Contribuyente
- **Periodo**: Periodo del adeudo
- **Monto**: Cantidad pendiente
- **Fecha de Vencimiento**: Plazo de pago
- **Días de Mora**: Días vencidos
- **Total General**: Sumatoria de adeudos

## Instrucciones de Uso
1. Seleccione los criterios de filtrado
2. Defina el periodo a reportar
3. Elija el tipo de agrupación
4. Genere el reporte
5. Revise totales y subtotales
6. Exporte o imprima según necesidad

## Notas Importantes
- Reporte en tiempo real
- Permite múltiples criterios de filtrado
- Información detallada por contrato
- Útil para cobranza y análisis
- Puede exportarse para análisis externo

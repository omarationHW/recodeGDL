# Rpt_Contratos

## Descripción
Reporte general de contratos del sistema de aseo contratado.

## Funcionalidades
- Listado completo de contratos
- Filtrado por estatus, zona, tipo
- Información detallada por contrato
- Agrupaciones diversas
- Exportación de datos
- Impresión de reportes

## Campos
- **Número de Contrato**: Identificador
- **Nombre/Razón Social**: Titular
- **Domicilio**: Ubicación del servicio
- **Tipo de Servicio**: Clasificación
- **Zona**: Área geográfica
- **Estatus**: Vigente, cancelado, suspendido
- **Fecha de Alta**: Registro en el sistema
- **Periodicidad**: Frecuencia de pago

## Instrucciones de Uso
1. Seleccione los criterios de filtrado
2. Defina el tipo de agrupación
3. Genere el reporte
4. Revise la información
5. Exporte o imprima según necesidad

## Notas Importantes
- Vista completa del padrón
- Útil para análisis y estadísticas
- Puede filtrarse por múltiples criterios
- Exportable a Excel/PDF
- Base para toma de decisiones

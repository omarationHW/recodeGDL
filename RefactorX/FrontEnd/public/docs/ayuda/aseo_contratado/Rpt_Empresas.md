# Rpt_Empresas

## Descripción
Reporte del catálogo de empresas del sistema de aseo contratado.

## Funcionalidades
- Listado de empresas registradas
- Información detallada de cada empresa
- Filtros de búsqueda
- Exportación de datos
- Impresión de reportes

## Campos
- **Clave de Empresa**: Identificador único
- **Nombre o Razón Social**: Denominación
- **RFC**: Registro Federal de Contribuyentes
- **Domicilio**: Ubicación
- **Teléfono**: Contacto
- **Email**: Correo electrónico
- **Estatus**: Activa/Inactiva

## Instrucciones de Uso
1. Acceda al módulo de reportes
2. Seleccione filtros si es necesario
3. Genere el reporte
4. Revise la información de empresas
5. Exporte o imprima según necesidad

## Notas Importantes
- Información del catálogo maestro
- Útil para auditoría y control
- Puede filtrarse por estatus
- Exportable a diferentes formatos
- Base para otros reportes del sistema

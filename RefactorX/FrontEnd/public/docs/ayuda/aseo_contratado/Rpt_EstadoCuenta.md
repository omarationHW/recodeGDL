# Rpt_EstadoCuenta

## Descripción
Módulo para la generación de estados de cuenta de contratos del sistema de aseo contratado.

## Funcionalidades
- Generación de estado de cuenta detallado
- Consulta de movimientos por periodo
- Desglose de adeudos y pagos
- Cálculo de saldos
- Impresión de documentos
- Exportación a PDF

## Campos
- **Número de Contrato**: Identificador del contrato
- **Periodo**: Rango de fechas para el reporte
- **Adeudos**: Cargos generados
- **Pagos**: Abonos realizados
- **Saldo**: Pendiente de pago
- **Detalle de Movimientos**: Historial completo

## Instrucciones de Uso
1. Ingrese el número de contrato
2. Seleccione el periodo a consultar
3. Genere el estado de cuenta
4. Revise el detalle de movimientos
5. Verifique los saldos
6. Imprima o exporte el documento

## Notas Importantes
- Información en tiempo real
- Incluye todos los movimientos del periodo
- Desglose detallado de cargos
- Puede generarse para cualquier fecha
- Documento oficial para trámites

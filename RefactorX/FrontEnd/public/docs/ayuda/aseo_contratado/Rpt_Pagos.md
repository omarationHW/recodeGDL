# Rpt_Pagos

## Descripción
Reporte de pagos aplicados en el sistema de aseo contratado.

## Funcionalidades
- Listado de pagos registrados
- Filtrado por periodo, contrato, recaudadora
- Desglose de conceptos pagados
- Totales por forma de pago
- Exportación de datos
- Conciliación bancaria

## Campos
- **Fecha de Pago**: Cuando se realizó
- **Número de Contrato**: Contrato que pagó
- **Nombre**: Contribuyente
- **Monto Pagado**: Cantidad
- **Forma de Pago**: Efectivo, cheque, transferencia
- **Recaudadora**: Quien recibió el pago
- **Folio de Recibo**: Comprobante

## Instrucciones de Uso
1. Seleccione el rango de fechas
2. Aplique filtros adicionales si necesita
3. Genere el reporte
4. Revise los totales por forma de pago
5. Exporte para conciliación si es necesario
6. Imprima el reporte

## Notas Importantes
- Información en tiempo real
- Útil para arqueos de caja
- Base para conciliación bancaria
- Puede agruparse por diferentes criterios
- Incluye todos los conceptos pagados

# ABCPagosxfol - Pagos por Folio

## Propósito Administrativo
Módulo alternativo para registro de pagos cuando se conoce directamente el folio RCM, agilizando la captura sin necesidad de búsqueda previa.

## Funcionalidad Principal
Permite aplicar pagos de mantenimiento ingresando directamente el folio RCM, mostrando inmediatamente los adeudos pendientes y permitiendo seleccionar períodos a pagar.

## Datos y Tablas
Similar a ABCPagos pero optimizado para acceso directo por folio

## Usuarios del Sistema
Cajeros experimentados, ventanilla rápida

# MultipleNombre - Consulta Múltiple por Nombre

## Propósito Administrativo
Módulo de consulta que permite buscar espacios funerarios por nombre del concesionario, facilitando la localización de registros cuando no se conoce la ubicación física o el folio.

## Funcionalidad Principal
Realiza búsquedas por nombre completo o parcial del concesionario, mostrando todos los espacios registrados a ese nombre con su información completa.

## Datos y Tablas
**ta_13_datosrcm** - Búsqueda por campo nombre con LIKE

## Usuarios del Sistema
Personal de ventanilla, cajeros, supervisores

# TitulosSin - Títulos Simplificados

## Propósito Administrativo
Versión simplificada del módulo de títulos para impresión rápida sin validaciones extendidas, utilizada en casos especiales o reimpresiones.

## Funcionalidad Principal
Impresión de títulos con proceso simplificado, útil para casos excepcionales o cuando ya se validó la información previamente.

## Datos y Tablas
Similar a Titulos pero con proceso de validación reducido

## Usuarios del Sistema
Personal de títulos autorizado, supervisores

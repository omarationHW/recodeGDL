# TrasladoFolSin - Traslados Simplificados

## Propósito Administrativo
Módulo simplificado para registro de traslados internos sin complejidad adicional, utilizado para movimientos rutinarios dentro del mismo panteón.

## Funcionalidad Principal
Versión simplificada del módulo de traslados para movimientos internos rápidos, manteniendo el registro básico del movimiento.

## Datos y Tablas
**ta_13_traslados** - Registro simplificado de traslados

## Usuarios del Sistema
Personal de panteones, operadores de campo

# consulta400 - Consulta de Bases Históricas

## Propósito Administrativo
Módulo especializado para consultar registros de bases de datos históricas de sistemas anteriores (versión 400), permitiendo acceso a información migrada.

## Funcionalidad Principal
Proporciona acceso de solo lectura a bases de datos históricas del sistema anterior, permitiendo consultar información de espacios registrados en el sistema previo.

## Proceso de Negocio

### ¿Qué hace?
- Conecta a múltiples bases de datos históricas
- Consulta registros del sistema anterior (versión 400)
- Muestra información de pagos históricos
- Permite búsqueda por ubicación en sistema antiguo

### ¿Para qué sirve?
- Acceso a información histórica
- Consulta de pagos en sistema anterior
- Soporte en migración de datos
- Verificación de información antigua

## Datos y Tablas
**Bases de datos históricas**: cmf01dcem, cmf01pcem (sistema versión 400)

## Usuarios del Sistema
Supervisores, personal de archivo, administradores del sistema

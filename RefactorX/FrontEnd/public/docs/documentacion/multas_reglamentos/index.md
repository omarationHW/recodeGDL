# Índice de Documentación - Módulo Multas y Reglamentos

## Acerca de este Módulo
El módulo de **Multas y Reglamentos** gestiona todo el ciclo de vida de las multas municipales, incluyendo emisión, descuentos, pagos, requerimientos, convenios y procesos de cobro coactivo.

## Información del Proyecto

### Migración: Informix → PostgreSQL
Este módulo está siendo migrado desde:
- **Sistema Original:** Delphi + Informix
- **Sistema Objetivo:** Laravel + Vue.js + PostgreSQL

### Arquitectura Nueva
- **Backend:** Laravel 10+ con patrón eRequest/eResponse
- **Frontend:** Vue.js 3 (Composition API)
- **Base de Datos:** PostgreSQL 13+ con stored procedures
- **API:** Endpoint único `/api/execute` con actions dinámicas
